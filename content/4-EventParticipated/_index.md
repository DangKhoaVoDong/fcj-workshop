### [Event 5](4.5-Event5/)
&emsp;**Event Name:** Workshop: AWS Cloud Mastery Series #3

&emsp;**Time:** 08:30 - 12:00, November 29, 2025

&emsp;**Location:** AWS Vietnam Office

&emsp;**Role:** Attendee

&emsp;**Description:**
* Deep dive into the 5 security pillars according to AWS Well-Architected standards:
    1.  **Identity (IAM):** Access management, Zero Trust, MFA.
    2.  **Detection:** Continuous monitoring with CloudTrail, GuardDuty.
    3.  **Infrastructure Protection:** Network protection (VPC, WAF, Shield).
    4.  **Data Protection:** Data encryption (KMS) and secrets management.
    5.  **Incident Response:** Automated incident response processes.

&emsp;**Key Outcomes:**
* Understood the Shared Responsibility Model and common security threats.
* Learned how to establish multi-layer security architecture (Defense in Depth) and automate detection and response to security incidents on the Cloud.
