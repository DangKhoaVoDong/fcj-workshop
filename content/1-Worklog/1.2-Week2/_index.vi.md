### Kết quả đạt được tuần 2:

- Nắm được tổng quan về các dịch vụ AWS:

  - Hiểu 4 nhóm dịch vụ chính và công dụng của từng nhóm
  - Biết khi nào nên sử dụng dịch vụ nào
  - Làm quen với AWS Regions và Availability Zones

- Cài đặt và cấu hình thành công AWS CLI:

  - Cài đặt AWS CLI trên Windows
  - Tạo IAM User với quyền phù hợp
  - Cấu hình Access Key và Secret Key
  - Chạy thử các lệnh CLI cơ bản

- Thực hành với Amazon EC2:

  - Tạo thành công EC2 instance đầu tiên
  - Hiểu về Instance types và AMI
  - Cấu hình Security Group
  - Kết nối SSH vào EC2 instance thành công
