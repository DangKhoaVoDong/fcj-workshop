### Kết quả đạt được tuần 6:

- Nắm được về Load Balancing:

  - Hiểu Load Balancer là gì và lợi ích
  - Phân biệt ALB, NLB, CLB
  - Tạo Application Load Balancer thành công
  - Cấu hình target groups và health checks

  - Hiểu RDS là gì và lợi ích so với tự cài database
  - Biết các database engines AWS hỗ trợ
  - Tạo RDS MySQL instance thành công
  - Cấu hình security groups cho RDS
  - Kết nối từ EC2 vào RDS
  - Hiểu về RDS backups và snapshots
